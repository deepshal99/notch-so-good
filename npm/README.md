# Notch So Good

**The world's smallest coworker lives in your Mac's notch.**

Meet **Chawd**. He's a mass of pixels. He lives in your MacBook's notch. And he has one job: watch Claude Code so you don't have to.

## Install

```bash
npx notch-so-good
```

10 seconds. No Xcode, no dependencies, no sign-up.

## What Chawd Does

- **Watches Claude Code** — appears as a black pill extending your notch with a live timer
- **13 idle animations** — wave, dance, sneeze, peek-a-boo, yawn, hiccup, spin, stretch, and more
- **Follows your mouse** — tiny pixel eyes track your cursor
- **Gets drowsy** — falls asleep if you ignore him, startled jolt when you return
- **Notifies you** — notch expands when Claude needs input. Click to jump back to terminal
- **Multi-session** — hover the pill to see all active Claude sessions grouped by project

## Requirements

- macOS 14+ (Sonoma or later)
- Claude Code
- A MacBook with a notch (recommended)
